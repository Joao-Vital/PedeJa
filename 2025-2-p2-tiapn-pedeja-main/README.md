# PedeJá

`CURSO: Análise e Desenvolvimento de Sistemas`

`DISCIPLINA: Trabalho Interdisciplinar Aplicações para Processos de Negócios`

`2º semestre/2025`

O projeto PedeJá tem como propósito apresentar informações essenciais sobre o parceiro envolvido, possibilitando uma compreensão mais ampla da proposta enquanto iniciativa extensionista. O parceiro em questão é a empresa Bartô Gastronomia, instituição que, além de administrar uma hamburgueria artesanal, promove uma oficina gastronômica para a comunidade local.

Nosso objetivo é desenvolver um website institucional e comercial que supere as limitações do sistema atual e fortaleça a presença digital da organização. A nova plataforma terá a função de facilitar as vendas, valorizar a trajetória da associação, divulgar oficinas, além de compartilhar informações que incentivem doações e ampliem o engajamento comunitário.

## Instruções de utilização

A primeira versão do  nosso sistema já está disponível
Você poderá acessá-lo através desse link: https://2025-2-p2-tiapn-pedeja.vercel.app/index.html.
1. Página Inicial

Ao acessar a página inicial do sistema, o usuário encontrará informações sobre as oficinas e os produtos do parceiro. Ao final da página, há dois botões que redirecionam para as páginas de Cardápio e Oficinas. Ressalta-se que, para acessar essas funcionalidades, é necessário que o usuário esteja devidamente logado em sua conta.

2. Cadastro de Usuário

Para realizar o cadastro de uma conta de cliente, o usuário deve clicar no canto superior esquerdo da página inicial, no botão identificado como “Cadastrar”. Em seguida, será direcionado a uma página onde deverá informar seu e-mail e definir uma senha de acesso. Após o preenchimento dos dados, deve-se clicar no botão “Cadastrar”, concluindo assim a criação da conta. O sistema redirecionará automaticamente o usuário para a página de login.

Caso o usuário já possua uma conta, poderá selecionar a opção “Entrar”, não sendo necessário repetir o processo de cadastro.

3. Login no Sistema

Na página de login, o usuário deverá informar o e-mail e a senha cadastrados anteriormente. Caso deseje manter o acesso ativo mesmo após fechar o navegador, poderá selecionar a opção “Lembrar-me”. Após o preenchimento dos campos, deve-se clicar no botão “Entrar”.

Se o usuário acessar essa página sem possuir cadastro prévio, poderá utilizar a opção “Cadastrar-se” para criar sua conta.

4. Cadastro de Perfil

No primeiro acesso ao sistema, após o login, será solicitado que o usuário realize o cadastro do perfil. Esse cadastro é necessário para a inscrição em oficinas e para a realização de pedidos no estabelecimento.

Na página de cadastro de perfil, é obrigatório o preenchimento dos seguintes campos: nome, sobrenome, CPF, CEP, endereço e número. Opcionalmente, o usuário poderá informar complemento do endereço, telefone, Facebook e Instagram. Após o preenchimento dos dados, deve-se clicar no botão “Salvar Perfil” para concluir o processo.

5. Cardápio

Após o salvamento do perfil, o usuário será redirecionado automaticamente para a página de cardápio. Nessa seção, estarão disponíveis os produtos do parceiro, organizados nas categorias Hambúrgueres, Bebidas e Acompanhamentos. Para realizar um pedido, o usuário deve selecionar os itens desejados e clicar na opção “Fazer Pedido”.

6. Oficinas

Para acessar a página de oficinas, o usuário deve selecionar a opção “Oficinas”, localizada na parte superior do site. Nessa página, é possível realizar a inscrição nas oficinas disponíveis, favoritar oficinas de interesse e visualizar aquelas que já foram concluídas.
